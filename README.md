# Sales Performance & Customer Behavior Analysis

## Overview
This project analyzes fashion retail sales data to identify revenue drivers, product performance patterns, customer purchasing behavior, and payment preferences.

## Dataset
The dataset contains transactional sales records including product type, purchase amount, review ratings, purchase dates, and payment methods.

## Key Insights
- Revenue is concentrated among a small set of high-value products.
- High-frequency products differ from high-revenue products.
- Customer satisfaction shows no significant correlation with spending.
- Credit cards generate the largest share of revenue.
